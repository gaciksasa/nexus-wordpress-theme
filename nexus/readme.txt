Nexus is a premium multi-purpose WordPress theme with Elementor support, FSE/Gutenberg compatibility, WooCommerce integration, dark mode, and 15+ diverse demo sites.

== Description ==

Nexus is a fast, flexible, and fully-featured WordPress theme designed for businesses, creatives, and online stores. It ships with 15 professionally designed demo sites covering popular industries, and supports Elementor, native Gutenberg/FSE, and WooCommerce out of the box.

Key Features:
- 15+ one-click importable demo sites
- Elementor page builder support with 13+ custom widgets
- Full Site Editing (FSE) with theme.json
- WooCommerce ready with off-canvas cart, quick view, and wishlist
- Dark mode support with system preference detection
- Kirki Customizer Framework integration
- Custom Post Types: Portfolio, Team, Testimonials, Services
- Mega menu support
- RTL language support
- Translation ready (POT file included)
- WCAG 2.1 AA accessibility compliant
- PageSpeed optimized (90+ score)
- WPML compatible

== Installation ==

1. Upload the `nexus` folder to your `/wp-content/themes/` directory.
2. Activate the theme from the WordPress Themes screen (Appearance > Themes).
3. Follow the setup wizard to install recommended plugins and import a demo.

Child Theme:
A child theme is included in the package (`nexus-child`). It is strongly recommended to use the child theme when customizing the theme so your changes survive updates.

== Frequently Asked Questions ==

= Do I need Elementor to use this theme? =
No. Nexus fully supports native WordPress Gutenberg/FSE blocks. Elementor is an optional enhancement.

= Is WooCommerce required? =
No. WooCommerce is optional. The theme functions perfectly without it.

= Does Nexus include demo content? =
Yes. 15 demo sites are included. You can import them via the setup wizard (Appearance > Nexus Setup).

= How do I install the theme? =
Upload the nexus.zip file via Appearance > Themes > Add New > Upload Theme.

= Is the theme translation-ready? =
Yes. A .pot file is included in the `languages` folder.

== WooCommerce Template Overrides ==

The following WooCommerce templates are overridden in the `/woocommerce/` directory:

- woocommerce/archive-product.php (WooCommerce 8.6.0)
- woocommerce/single-product.php (WooCommerce 1.6.4)
- woocommerce/cart/mini-cart.php (WooCommerce 10.0.0)
- woocommerce/checkout/form-checkout.php (WooCommerce 9.4.0)
